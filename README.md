# Schreibfit
Github Repository für unser Projekt (Rechtschreibtrainer, Kooperation ITP/SEW)

Zu Beginn erscheint eine Anmelde-Seite, auf der lediglich überprüft wird, ob eine gültige TGM-Schüler-E-Mail-Adresse eingegeben wird. Ein echtes Benutzerkonto-System gibt es nicht, das Passwort dient nur als Platzhalter für ein mögliches zukünftiges Login-Konzept.

Nach dem Login gelangt man ins Hauptmenü. Dort kann man zwischen mehreren Funktionen wählen. Im Verwaltungsbereich lassen sich Fragen aus einer Textdatei laden, bearbeiten, löschen oder neu erstellen. Dabei gibt es zwei Fragetypen: Textfragen (z. B. Beschreibungen oder Lückentexte) sowie Bildfragen in Form von URLs.

Im Quizmodus werden zufällig ausgewählte Fragen angezeigt. Der Benutzer muss die richtige Schreibweise eingeben, welche anschließend automatisch überprüft wird. Je nach Schwierigkeitsstufe werden Punkte vergeben und am Ende wird eine Statistik über richtige und falsche Antworten angezeigt.

Zusätzlich gibt es einen Spielmodus in Form eines Wortsalats. Dabei werden Buchstaben eines Wortes gemischt dargestellt und müssen in die richtige Reihenfolge gebracht werden. Auch hier erfolgt eine Auswertung mit Punkten.

Das Ziel des Programms ist es, durch wiederholtes Üben die Rechtschreibkenntnisse zu verbessern und das Lernen spielerisch zu gestalten.
